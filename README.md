## Installation
```bash
python3 -m venv <your_venv>
source <your_venv>/bin/activate
mkdir <your_average_price_folder>
cd <your_average_price_folder>
git clone git@github.com:tibor-reiss/average_price.git .
pip3 install .
```
## Running
```bash
cd <your_venv>/lib/<python_version>/site-packages/average_price
python3 write_to_file.py
python3 main.py
```
